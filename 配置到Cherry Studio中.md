适用范围
用于将 Cherry Studio 接入兼容 Anthropic API 的第三方中转站、代理或网关。

开始前请确认你已获取：

API 密钥 ：进入后台，点击API密钥菜单
API 端点：进入后台，点击API密钥菜单
步骤1：打开 Cherry Studio 设置
打开设置
在左侧的模型平台列表点击添加
<img width="3600" height="2142" alt="image" src="https://github.com/user-attachments/assets/babf104f-af87-4092-a4a6-b09c175eee6b" />


步骤2：添加提供商
随便输入一个名称
提供商类型选择 Anthropic
点击确定
<img width="826" height="1132" alt="image" src="https://github.com/user-attachments/assets/145ee6ee-b624-4cba-83c9-419a353842e9" />


步骤3：输入API密钥和API地址
输入API密钥和API地址
点击获取模型列表
如果API可用，会出现模型列表，点击途中的添加所有，点击确定
然后关闭页面，返回首页
<img width="1009" height="368" alt="image" src="https://github.com/user-attachments/assets/d493cd9f-3f4b-406c-b723-adf312f5f383" />

<img width="1011" height="595" alt="image" src="https://github.com/user-attachments/assets/d13dc185-ccf8-4cdd-a48e-48a121466016" />




步骤4：验证是否成功
点击图片中的位置可切换模型
选择刚刚添加的模型
发送hi测试是否可用
